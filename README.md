# Projeto_COO

Projeto da disciplina de Computacao Orientada a Objetos para reestruturar, em Java, o codigo procedural de um jogo *shoot 'em up*.

## Objetivos

- Aplicar encapsulamento, heranca, composicao e uma hierarquia adequada de classes e interfaces.
- Substituir os arrays usados para gerenciar entidades por colecoes Java.
- Implementar um terceiro tipo de inimigo com aparencia e comportamento proprios.
- Documentar as decisoes de projeto em um relatorio.

## Estrutura

- `src/`: codigo-fonte Java.
- `docs/`: enunciado e documentacao do projeto.

## Situacao atual

Projeto em desenvolvimento. As classes originais `Main` e `GameLib` foram mantidas intactas ate o momento.
